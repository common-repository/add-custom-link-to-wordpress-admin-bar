Add the custom link to your wordpress admin bar

== Description ==

With the use of this plugin you can add custom link in your wordpress admin bar. You can also add child link of the parent link.

For more information or any query please <a href="http://sumitbansal.com/wordpress-plugin/add-custom-link-to-wordpress-admin-bar/">click here</a>

== Installation ==

Installation Instructions:

    Download the plugin and unzip it.
    Put the 'add_custom_link_to_wordpress_admin_bar' folder into your wp-content/plugins/ directory.
    Go to the Plugins page in your WordPress Admin and activate the Add custom link to wordpress admin bar
    Go to Add Custom Link->Manage Custom Link Tab to manage the link setting according to you.
    To Add more link go to Add Custom Link->Add Custom Link
